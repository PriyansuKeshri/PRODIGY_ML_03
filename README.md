# PRODIGY_ML_03  
## Cats vs Dogs Image Classification using SVM

This project is part of my **Machine Learning Internship at Prodigy InfoTech**.

The goal of this task is to build an **image classification model** that can identify whether an image contains a **cat or a dog**.

---

## 📌 Problem Statement

Implement a **Support Vector Machine (SVM)** model to classify images of cats and dogs using computer vision techniques.

---

## ⚙️ Technologies Used

- Python
- OpenCV
- NumPy
- Scikit-learn

---

## 📊 Machine Learning Model

**Support Vector Machine (SVM)**

Images were preprocessed and converted into numerical feature vectors before being used to train the SVM classifier.

---

## 🖼 Image Processing Steps

1. Load images using OpenCV  
2. Convert images to **grayscale**  
3. Resize images to **64×64**  
4. Flatten images into feature vectors  
5. Train SVM model for classification

---

## 📂 Dataset

Dogs vs Cats dataset containing labeled images of cats and dogs.

---

## 🚀 Conclusion

This project demonstrates how **Support Vector Machines can be applied to computer vision tasks** by converting images into numerical features and training a classification model.
